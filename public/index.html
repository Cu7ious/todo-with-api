Classic TodoMVC

Classic ToDo App

with React and Redux

Product requirements

  • Postman Collection as API server
  • A simple TODO application that allows to
    • Add new task
    • Edit existing task
    • List all tasks
    • Delete existing task

Code requirements

  • Prefer the jsx extension over capital letters
  • One react component per file
  • Avoid large render methods
  • Test your code very extensively

Libraries

  • React
  • Redux (with Ducks)
  • Webpack
  • Babel
  • Jest
  • Bootstrap
